grid-template-columns

Define las columnas de un contenedor de rejilla . Puede especificar el ancho de una columna usando una palabra clave (como auto) o una longitud (como 10px). El número de columnas está determinado por el número de valores definidos en la lista separada por espacios.

ACEPTA CUALQUIER TIPO DE UNIDAD DE MEDIDA: CM,%,PX,REM,FR


grid-template-columns: repeat( 6, 16% ); /Se usa repeat para formar varias columnas (el primer valor es la cantidad de columnas, el segundo valor es el tamaño de las columnas)


Tambien se puede usar el repeat con 3 valores (el primer valor es la cantidad de columnas, el segundo y el tercer valor es el tamaño de las columnas) en este caso el resultado seria de forma intercalada

EJEMPLO:
grid-template-columns: repeat( 3, 16% 1fr) DE ESTA FORMA EL RESULTADO SERIA INTERCALADO, 3 COLUMNAS DE 16% Y 3 COLUMNAS DE 1FR

Tambien hay una unidad de llamada FR
y sirve para dividir el espacio disponible en fracciones
