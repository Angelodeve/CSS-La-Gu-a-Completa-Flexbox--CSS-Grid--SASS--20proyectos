Primeros Pasos con Gulp e instalar Dependencias

  • npm init (Esto creara un archico package.jsones para poder instalar las dependencias)Con esto iniciamos el proyecto, nos aparecera algunos datos de nuestro proyecto que tenemos que completar
  • npm i --save-dev gulp (creara una carpeta (node_module)y el archivo package-lock.json ) con esto estamos instalando las dependencias de gulp
  • gulp -v con esto nos mostrara la version de nuestro gulp
  • despues creamos un archivo gulpfile.js en este archivo vamos a elaborar tareas de gulp
  • npm i si eliminamos las dependencias con este comando podemos instalar de nuevo las dependencias
  • npm i --save-dev sass gulp-sasscon este comando vamos a instalar dependencias sass, aqui va a instalar todo el concocimiento de sass(sabe que es una variables,su sintaxis) y este de aca nos sirve para poderconectar gulpfile.js y app.scssgulp-sass
  • npm i --save-dev autoprefixer gulp-postcssCon estas dos dependencias podemos escribir codigo de ultima generacion css y nos va a crear versiones compatibles que no la soportan
